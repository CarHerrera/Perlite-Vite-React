# Welcome

This is a website that uses the open-source Perlite to host my notes online.  Perlite does not support excalidraw out of the box and this was a plugin that I felt was necessary for my notes. To make excalidraw work with Perlite, I used Vite and React to host all the modules that are needed for excalidraw. I used a vite-php  to use vite as a backend-integration and continued using perlite as the host for my notes.

## Bases

Check out Movies.base! It currently holds over 3000 movies that were found from a kaggle database. I turned all the entries into MD notes and added some YAML tags based on the dataset. You are able to filter the notes based on those yaml tags!
It is currently in a proof of concept stage, and I do plan to add more functionality to the bases. 
## Excalidraw

This was what originally made me want to create this project in the first place. I loved using Excalidraw while I was a student at the University of Maryland. I integrated it into perlite and it can load custom excalidraw notes. Check it out under the Excalidraw folder.
